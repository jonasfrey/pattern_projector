Template Projector Application - Complete Technical Requirements
📋 Project Overview
A production-ready, single-page web application that projects sewing patterns onto a table surface using a ceiling-mounted projector. Users can upload vector sewing patterns, calibrate the projection size using a physical tape measure, and lock the calibration for accurate pattern tracing. Built with Deno.js backend and vue js frontend.

🎯 Core Requirements
1. Application Purpose
Primary Goal: Project sewing patterns from a ceiling-mounted projector onto a table

Key Functionality: Calibrate projection size to match physical measurements

Workflow: Upload pattern → Display on table → Calibrate with tape measure → Lock calibration → Trace pattern

2. Architecture
Backend (Deno.js)
Deno.js serves as the main application server

File upload and management

Project state storage (JSON)

SVG/PDF vector processing

WebSocket for real-time updates

HTTP/2 for static assets and API endpoints

Frontend (vue js)
Single Page Application (SPA)

vue js for reactive UI components

Canvas/SVG rendering for pattern display

Vanilla CSS with CSS Grid/Flexbox for layout

Drag and drop for file upload

Calibration Logic

// src/calibration/manager.ts
export class CalibrationManager {
    private patternScale: number = 1.0;
    private referenceDistance: number = 10; // cm
    private tolerance: number = 0.5; // mm
    private locked: boolean = false;
    
    /**
     * Calibrate pattern to match physical tape measure
     * @param projectedDistance - Distance on projected pattern in pixels
     * @param actualDistance - Actual distance on table in cm
     */
    calibrate(projectedDistance: number, actualDistance: number): number {
        if (this.locked) {
            throw new Error('Calibration is locked');
        }
        
        const newScale = (actualDistance / projectedDistance) * this.patternScale;
        this.patternScale = newScale;
        
        // Log calibration
        console.log(`Calibrated: ${projectedDistance}px = ${actualDistance}cm`);
        console.log(`New scale: ${newScale.toFixed(3)}x`);
        
        return newScale;
    }
    
    lock(): void {
        this.locked = true;
        console.log('Calibration locked');
    }
    
    unlock(): void {
        this.locked = false;
        console.log('Calibration unlocked');
    }
    
    getCalibrationStatus(): CalibrationStatus {
        return {
            scale: this.patternScale,
            referenceDistance: this.referenceDistance,
            locked: this.locked,
            accuracy: this.calculateAccuracy()
        };
    }
    
    private calculateAccuracy(): number {
        // Calculate accuracy based on current scale
        // Returns accuracy in mm
        return 0.2; // Placeholder
    }
}

🚀 Installation & Setup
Auto-Installation with deno task start

// tasks/install.ts
export async function installDependencies() {
    console.log('🔧 Installing required dependencies...');
    
    // Check and install ImageMagick for SVG/PDF processing
    const magickInstalled = await checkCommand('convert');
    if (!magickInstalled) {
        console.log('📦 Installing ImageMagick...');
        await installImageMagick();
    }
    
    // Check and install librsvg for SVG rendering
    const rsvgInstalled = await checkCommand('rsvg-convert');
    if (!rsvgInstalled) {
        console.log('📦 Installing librsvg...');
        await installRsvg();
    }
    
    // Install Deno dependencies
    console.log('📦 Installing Deno dependencies...');
    await installDenoDeps();
    
    console.log('✅ All dependencies installed!');
}

async function installImageMagick() {
    const os = Deno.build.os;
    switch(os) {
        case 'linux':
            await exec('sudo apt update && sudo apt install -y imagemagick');
            break;
        case 'darwin':
            await exec('brew install imagemagick');
            break;
        case 'windows':
            await exec('choco install imagemagick');
            break;
    }
}

async function installRsvg() {
    if (Deno.build.os === 'linux') {
        await exec('sudo apt install -y librsvg2-bin');
    } else if (Deno.build.os === 'darwin') {
        await exec('brew install librsvg');
    }
}

🎯 Special Considerations
Calibration Accuracy
The calibration must be accurate to within ±0.5mm

Consider using a calibration grid or known reference objects

Provide visual feedback during calibration (color changes, alerts)

Projection Environment
Dark theme is essential for projection visibility

High contrast patterns (white on dark background)

Consider projector resolution and aspect ratio

Pattern Formats
SVG: Preferred format (scalable, editable)

PDF: Common format for sewing patterns

DXF: Industry standard for CAD patterns

AI: Adobe Illustrator format

Locking Mechanism
Prevent accidental clicks from changing calibration

Visual indicator when locked (padlock icon, color change)

Require confirmation to unlock
